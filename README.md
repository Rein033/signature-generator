# Email Signature Generator

Simpele webtool om een e-mail handtekening te maken en te kopiëren naar Outlook of Gmail.

## Gebruik

1. Open de website
2. Vul je naam, functie, mobiel en e-mail in
3. Klik op **Genereer**
4. Klik op **Kopieer**
5. Plak in Outlook of Gmail handtekening instellingen

## GitHub Pages hosten

1. Upload `index.html` naar deze repository
2. Ga naar **Settings → Pages**
3. Kies branch: `main`
4. Opslaan

Je site staat dan live op:
https://jouwgebruikersnaam.github.io/repositorynaam/

## Aanpassen

Open `index.html` en wijzig tekst of logo indien nodig.
